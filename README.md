# Spicy King Restaurant

A simple, modern, and responsive restaurant website built with **HTML**, **CSS**, and **Font Awesome** icons. This project is designed to showcase a restaurant's menu, about section, contact details, and footer with social media links. Perfect for learning front-end web development and responsive design.
